Welcome to HouseHunter

HouseHunter is a project created by Alex Buckley to make house hunting easier in Gig Harbor, WA.

I've scraped publicly-available data about ~ 17,000 single family homes from Pierce County property records. The goal of this project is to create a searchable database about prospective houses to buy.

The source code is available on my Github
